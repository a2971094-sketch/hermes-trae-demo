## 🔧 任务: 用户认证模块

### 分支
feat/user-auth

### 需求描述
实现 JWT 登录/注册/刷新三个 API 接口

接口:
- POST /api/auth/register — 注册 (username, password) → 返回 user_id
- POST /api/auth/login — 登录 (username, password) → 返回 access_token, refresh_token
- POST /api/auth/refresh — 刷新 token (refresh_token) → 返回新的 access_token

技术要求:
- FastAPI + SQLAlchemy + python-jose
- User 模型: id, username, password_hash, role, created_at
- Access token 30分钟过期, Refresh token 7天过期
- 密码用 bcrypt 加密
- 用内存 SQLite 数据库（便于测试）
- 单元测试覆盖三个接口

### 技术约束
- 框架: FastAPI
- 测试: pytest, 覆盖率 > 80%
- 代码风格: Black + ruff
